Removes all content in a WP site with just 1 click. 

== Description ==

The Void is ideal for test sites. This plugin deletes all content in a Wordpress single site. It deletes posts, pages, tags, categories, custom post types, taxonomy terms and menus.
No filters or parameters. The Void cleans your site with just one click. 

== Installation ==

1. Upload The Void to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the tools menu you will find a new item call `The Void`
